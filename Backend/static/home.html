How to Get Your API Keys

MongoDB URI:

  1. Visit MongoDB Atlas
  2. Create a free account or sign in
  3. Create a new cluster (free tier available)
  4. Click "Connect" on your cluster
  5. Choose "Connect your application"
  6. Copy the connection string
  7. Replace <password> with your database password

Example URI format:

mongodb+srv://username:password@cluster.xxxxx.mongodb.net/database?retryWrites=true&w=majority

Gemini API Key:

  1. Visit Google AI Studio
  2. Sign in with your Google account
  3. Click "Create API key" button
  4. Copy your new API key

Note: Keep your API key secure and never share it publicly

HuggingFace API Key:

  1. Go to HuggingFace Tokens
  2. Sign in to your account (create one if needed)
  3. Click "New token" button
  4. Give your token a name and select permissions
  5. Copy your new token (starts with 'hf_')

Note: Save your token immediately as it won't be shown again

Format: mongodb://username:password@host:port/database
